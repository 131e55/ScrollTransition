## :cd: Installation

### CocoaPods

ScrollTransition is available through [CocoaPods](http://cocoapods.org).

```ruby
platform :ios, '8.0'
use_frameworks!

target 'YOUR_TARGET_NAME' do
  pod 'ScrollTransition'
end
```
